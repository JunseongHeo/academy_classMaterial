B-2조 <개발멸망전> (전 개발하는 녀석들)

-조장 : 이재혁
-조원 : 양지선, 윤형진, 이정우, 나정수


-화면구현 프로젝트 주제 : [ 한정판 신발 거래(리셀) 사이트 ]

-주제 선정 이유 : 
 현재 MZ 세대의 재테크로 주목받고 있는 리셀테크(리셀+재테크) 항목 중 한정판 신발 거래를 위한 사이트를 개발.
여러 사이트에 분포된 상품 정보를 통합하여 가격 비교, 구매, 판매를 한번에 가능하도록 함.

-향후 가능성 :
 낮은 자산으로 투자 가능한 리셀 시장이 커짐에 따라 여러 항목으로 확대 가능.
 인터넷 사이트 뿐만 아니라 앱 등 여러 플랫폼을 통해 사용자 유입 가능.
 정보 게재 혹은 거래 수수료 등으로 수익 창출 가능.


-역할 분담
: 이재혁 ( 발표 보조, ppt 보조, 코딩 및 화면 구현 보조 )
: 양지선 ( ppt 제작, 코딩 및 화면 구현 보조 )
: 윤형진 ( 코딩 및 화면 구현, ppt 보조 )
: 이정우 ( 코딩 및 화면 구현, ppt 보조 )
: 나정수 ( 발표, ppt 보조)
